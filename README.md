## EMD_Android_APK
Android basic source (BASIC!, Laughton, 2011) to simulate parts of EMD Procedure implemented by Schrausser et al. (1994, 2022, respectively). Android versions 4.0 or later.

### Files

- Source: emd.bas
- Include: strg.txt
- Install: EMDv4.2.0.apk
